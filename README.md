# aiCEO — Copilote CEO

Produit de copilotage exécutif personnel. Deux faces aujourd'hui qui convergent vers un produit unifié :

- **App Web** (`01_app-web/`) — cockpit Twisty HTML/CSS/JS, vitrine UX
- **MVP Node** (`03_mvp/`) — serveur Express local, arbitrage Claude, ingestion Outlook

**Pour comprendre où on en est, commencer par [`00_BOUSSOLE/`](./00_BOUSSOLE/)**

```

01_app-web/        ← cockpit Twisty statique

04_docs/           ← stratégie produit, benchmark, architecture

_drafts/           ← en maturation, non stabilisé
_archive/          ← versions figées, ne plus toucher

```
