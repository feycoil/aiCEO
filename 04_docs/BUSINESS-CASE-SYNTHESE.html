TL;DR — investissement plateforme + écosystème

aiCEO est un copilote stratégique pour CEO de PME qui transforme la charge mentale du dirigeant en arbitrage matinal de 5 minutes. MVP v0.4 livré le 24 avril 2026 (28/28 tâches arbitrées, ≈ 1 ct/jour de coût LLM).

Trajectoire 18 mois : v0.5 fusion (T2 2026) → V1 cloud proactif (T3-T4 2026) → V2 multi-tenant équipe (T1-T2 2027) → V3 coach + mobile (T3-T4 2027). Coût total ≈ 1,69 M€, financé fonds propres ETIC + appoint éventuel business angel sur la phase V2.

Modèle économique : dogfood interne ETIC sur v0.5 → V1 (gain de temps CEO valorisé, pas de revenue), puis SaaS premium 300-800 €/CEO/mois à partir de V2 sur 1-3 CEO pairs ETIC, ouverture sélective fin 2027. Le projet est pensé comme un investissement plateforme réutilisable, pas comme un startup play classique.

Lecture interne ETIC — point mort plateforme

v0.5 → V1 — usage interne, pas de revenue

Le bénéfice économique est mesuré par le gain de temps CEO Feycoil (45-65 k€/an de valeur reproductible à partir de V1) et la capitalisation sur la plateforme + design system Twisty réutilisable par l'agence ETIC.

Le seul gain Feycoil rembourse environ 13-18 % des coûts V1 annualisés. Le reste est un investissement R&D ETIC (modèle agence-startup classique).

V2+ — point mort opex vs total

Coût marginal CEO additionnel post-V2 ≈ 415-845 €/mois en early V2 (LLM + infra + support fondateur), descendant à 100-200 €/mois en V3 standardisé.

À 600 €/mois, le point mort cash sur opex est atteint dès le 2-3ème CEO pilote. Couverture du coût total trajectoire (≈ 1,75 M€ tout inclus) suppose 250-350 CEO clients, horizon réaliste 2029-2030.

Risques connus & mitigations — vue priorisée
P0

Pont jetable v0.5 — risque produit

v0.5 local-first (SQLite + Outlook COM + Service Windows) sera intégralement re-architecturée en V1 cloud. Une partie du code sera jetée.

Mitigation — schéma SQLite 13 tables conçu pour migrer 1:1 vers Postgres ; couche d'abstraction services/outlook.js remplaçable par msal-node Graph sans toucher l'agent ; documentation explicite des morceaux jetables vs capitalisés (SPEC-TECHNIQUE-FUSION §13).

PRODUIT
P0

Modèle économique externe non validé — risque marché

Le pricing 300-800 €/CEO/mois repose sur des comparables (Welcome to the Jungle Solo 199 €, Pennylane Premium 320 €, Spendesk 400 €) et une intuition métier, pas sur 5+ entretiens validation acheteur ni sur un test landing page ouvert.

Mitigation — vague entretiens validation acheteur pré-V2 (10-15 CEO PME francophones hors écosystème ETIC) ; pilote V2 réel facturé sur 1-3 CEO pairs avec NPS mesuré M+3 / M+6 / M+12 ; option pivot outil interne ETIC standalone si la bande basse 300 €/mois ne tient pas.

MARCHÉ
P1

Dépendance Anthropic — risque technique

v0.4 → V1 reposent sur Claude Sonnet 4.5 via API directe Anthropic. Pricing × 5, coupure ou variation qualité = produit compromis.

Mitigation — V1 mirroring AWS Bedrock EU Frankfurt (même modèle, autre canal contractuel) ; V2 LiteLLM comme couche multi-vendeur (Anthropic + Bedrock + Mistral + GPT fallback) ; prompts versionnés 02_design-system/prompts/ pour re-tuning rapide.

TECHNIQUE
P1

CEO pilote unique — risque opérationnel

Dogfood v0.4 → V1 entièrement porté par Major Fey. Indisponibilité ≥ 4 sem ou pivot de focus = boucle feedback cassée.

Mitigation — objectif COPIL T4 2026 : identifier 2 CEO pilotes additionnels écosystème ETIC pour entrée V2 ; rituel dogfood documenté (revue hebdo + ADR systématiques) reprenable par un autre CEO ; lead produit dédié provisionné dans budget V2.

OPÉRATIONNEL
P1

Écart compétences cloud / multi-tenant — risque équipe

v0.5 livrable par 2-3 ETP. V2 exige 8 ETP avec compétences cloud + Postgres RLS + multi-tenant + SOC 2 + DevOps : profils rares et chers en France.

Mitigation — sourcing CTO/lead infra dès go V1 (mi-2026) ; surcoût 12-15 % déjà intégré (900 €/jour, haut du marché senior) ; partenariat externalisé Padok / Theodo Cloud pour SOC 2 et DevOps.
