## Nouveautés

- **Assistant chat live (S4.01 + S4.02)** : table SQLite `assistant_conversations` + `assistant_messages` (FK CASCADE), 4 routes `/api/assistant/*`, **streaming Claude via SSE** (`messages.stream` Anthropic SDK), fallback démo offline chunk-by-chunk, frontend `assistant.html` (369 lignes) avec sidebar conversations + thread + indicateur typing animé + Ctrl+Enter + auto-resize textarea
- **groupes.html migré API (S4.03)** : `GET /api/groups` + drill-down per `GET /api/groups/:id`, cards portefeuille avec progression moyenne calculée client-side, KPIs (projets actifs/total, hot, avancement), drawer projets, palette Twisty rotative (lilac/coral/sage/amber)
- **projets.html migré API (S4.04)** : liste cross-groupe + filtres groupe (chips) + recherche full-text via `?q=`, badges status (hot/new/active/archived), pré-filtre via `?group=xxx` URL
- **projet.html template paramétré (S4.05)** : 1 template au lieu de 7-10 pages projet statiques, lit `?id=xxx`, fetch parallèle `Promise.all` (project + tasks + decisions), 5 KPIs (tâches ouvertes / en retard / décisions ouvertes / contacts / RDV), 2 listes (tâches + décisions)
- **contacts.html migré API (S4.06)** : cartes avec avatar initiales + filtres trust_level (haute/moyenne/basse), recherche full-text via `?q=`, mailto direct
- **decisions.html migré API + IA recommend (S4.07)** : registre filtrable par status, transitions inline (decide/execute/abandon), bouton « 💡 Recommander une suite » qui réutilise le streaming `/api/assistant/messages` avec prompt contextualisé (décision + statut + projet)
- **Raccourci desktop "Cockpit aiCEO" (S4.08)** : `scripts/service-windows/install-desktop-shortcut.ps1` (install/uninstall/status), icône custom si présente sinon fallback Edge/Chrome, port configurable
- **Polish service Windows (S4.09)** : `start-aiCEO-at-logon.ps1` étendu avec rotation des logs (rotate à 2 Mo, 3 archives max ≈ 8 Mo total) — KISS pré-S5, pas de winston
- **INSTALL-WINDOWS.md consolidé (S4.09)** : 9 sections (pré-requis, démarrage manuel, autostart Variante D, raccourci Bureau, sync Outlook, recette validation, mises à jour, désinstallation complète, troubleshooting 8 cas)
