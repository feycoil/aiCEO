## Architecture finale

### Backend
- Express + node:sqlite (Node 24)
- 14 routers REST CRUD : `tasks`, `decisions`, `contacts`, `projects`, `groups`, `events`, `cockpit`, `arbitrage`, `evening`, `weekly-reviews`, `big-rocks`, `system`, `preferences`, `assistant`, `knowledge`
- SSE bus EventEmitter + heartbeats
- 4 routes assistant streaming via `messages.stream` Anthropic SDK (mode démo si `ANTHROPIC_API_KEY` absent)
- Headers sécurité HTTP (S6.5)

- DS partagé `_shared/` (CSS tokens + JS bind + sprite icônes + fonts)
- 19 bind-*.js dans `_shared/`
- PWA installable
- 13/17 pages câblées API + 4 pages static (hub, aide, components, helpers) + 3 preview v0.7/v0.8
