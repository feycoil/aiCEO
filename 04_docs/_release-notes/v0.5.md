## Architecture livrée

### Backend
- Express + node:sqlite (Node 24)
- 11 routers REST CRUD (`/api/{tasks,decisions,contacts,projects,groups,events,cockpit,arbitrage,evening,weekly-reviews,big-rocks,system,assistant}`)
- SSE streaming via EventEmitter bus + `GET /api/cockpit/stream` (S3.05)
- Migrations versionnées (`data/migrations/*.sql`)
- Isolation tests via `AICEO_DB_OVERRIDE`
- Mode démo automatique sans clé Anthropic (fallback offline)

### Frontend (12 pages)
- **Cockpit** (`/`) : 5 cards intention + big rocks + tâches + décisions + alertes
- **Arbitrage** (`/arbitrage`) : 3 colonnes drag-drop natif HTML5
- **Bilan soir** (`/evening`) : humeur + énergie + top3 + streak persistant
- **Tâches** (`/taches`) : Eisenhower 2×2 + inbox + quick-add
- **Agenda** (`/agenda`) : grille hebdo lun-dim + drag-drop tâche → date
- **Revues** (`/revues`) : 12 dernières semaines + Big Rocks max 3 + auto-draft Claude
- **Groupes** (`/groupes`) : portefeuille avec drill-down drawer
- **Projets** (`/projets`) : liste cross-groupe + filtres + recherche
- **Projet** (`/projet?id=xxx`) : template paramétré (5 KPIs + tâches + décisions)
- **Contacts** (`/contacts`) : cards avatar + recherche + filtres trust
- **Décisions** (`/decisions`) : registre + transitions + IA recommend (réutilise streaming assistant)
- **Assistant** (`/assistant`) : chat live SSE chunk-par-chunk + sidebar conversations

### Intégrations
- **Outlook** : sync emails + calendrier toutes les 2h via schtasks (PowerShell + COM)
- **Anthropic Claude** : SDK officiel, `messages.stream` pour assistant + `messages.create` pour arbitrage / délégation / draft revue
- **GitHub** : 156 issues (114 ouvertes au cleanup + 42 fermées en livraison v0.5), 5 milestones fermés, 5 tags posés, 5 GitHub Releases publiées (post fix-consistence)

### Infrastructure Windows
- **Variante D Startup folder** : `aiCEO-Server.lnk` lance le serveur au logon (sans admin)
- **Raccourci Bureau** : `Cockpit aiCEO.lnk` ouvre le cockpit dans le navigateur par défaut
- **Sync Outlook schtasks** : `aiCEO-Outlook-Sync` toutes les 2h
- **Rotation logs** : 2 Mo / 3 archives via wrapper PowerShell (KISS pré-V1)
