    📚 Acronymes les plus utilisés

    • ADR · Architecture Decision Record · décision technique tracée
    • ADD-AI · AI-Driven Development · méthode où l'IA code, le CEO valide
    • KPI · Key Performance Indicator · métrique chiffrée suivie
    • LLM · Large Language Model · ChatGPT, Claude, etc.
    • MCP · Model Context Protocol · standard Anthropic pour connecter outils
    • MVP · Minimum Viable Product · version minimale qui marche
    • NPS · Net Promoter Score · satisfaction utilisateur 0-10
    • PWA · Progressive Web App · web installable comme app native
    • SaaS · Software as a Service · logiciel hébergé chez fournisseur
    • SSE · Server-Sent Events · streaming serveur → navigateur
    • UX/UI · User Experience / Interface · parcours + visuel

    🎯 Jargon interne aiCEO

    • Big Rocks · 3 priorités hebdo du CEO (gros cailloux)
    • Cockpit · page d'accueil de l'app aiCEO
    • Boucle du soir · rituel quotidien (humeur, top 3, streak)
    • Eisenhower 5 verdicts · Faire / Déléguer / Décaler / Archiver / Décliner
    • Inbox-zero · objectif arbitrage matinal — emails traités
    • Pin to knowledge · épingler dans la base mémoire long-terme
    • Spike · sprint court de validation d'hypothèse
    • Subagent · Claude spécialisé (architect, dev, designer, etc.)
    • Tool calling · LLM appelle fonction externe au lieu de juste générer texte
    • Twisty · nom du Design System aiCEO
    • j-binôme · journée de travail CEO + Claude (pas Claude seul)
