    👤 Actions à exécuter côté CEO (Windows)

    Tâches que Claude ne peut pas faire et qui requièrent ton intervention

    1. Push vers origin :
      git push origin main — pousse les commits S6.8 + audit + méthode
    2. Lancer la routine de mise à jour pilotage :
      pwsh -File update-pilotage.ps1 — régénère ce dashboard à la demande

      pwsh -File update-pilotage.ps1 -InstallHook — pilotage régénéré à chaque commit
    4. Restart serveur après tout patch backend :
      pwsh -File restart-server.ps1
    5. Valider l'ADR v8 ADD-AI dans DECISIONS.md (changer Statut "Proposé" → "Acté")
    6. Créer milestone GitHub Phase-1-Fondations + 5 issues S6.9 à S6.13
