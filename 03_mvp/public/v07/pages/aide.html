01Vue d ensemble

aiCEO est votre Compass executive local pour CEO francophone CSP+. Il ne remplace pas vos Decisions, il les structure et les protege : un Cockpit le matin pour arbitrer ce qui compte (Triage), des automatismes pour digerer la masse d emails, un Bilan le soir pour clore proprement, un Weekly Sync pour ajuster.

Philosophie : 100 % local, SQLite mono-instance, zero cloud applicatif. Vos donnees restent sur votre poste. Anthropic Claude sert uniquement au raisonnement (chat assistant + 4 routes LLM frontend) - aucune donnee de votre Hub n est conservee chez Anthropic.

Les 4 cycles (Sync)

  • Triage matin (5-10 min) : tri emails + Decisions du jour
  • Cockpit journee : KPIs, Pulse, Top 3 du jour, North Star
  • Bilan soir (3 min) : mood, energie, top 3 fait, streak
  • Weekly Sync vendredi (30 min) : retrospective + Big Rocks + auto-draft Claude

0312 termes canoniques (voix exec moderne)

Le glossaire complet est dans 04_docs/_design-v08-intentions/GLOSSAIRE-AICEO.md. Resume :

Architecture

  • Hub - votre espace top-level + page d accueil personnelle (fusion semantique)
  • Stream - vertical de business (admin / fi / op / r&d / maintenance / rh)
  • Project - entite metier principale (auto-status alerte/a-surveiller/sain)
  • Action - travail concret (matrice Eisenhower)
  • Decision - choix trace (4 statuts : ouverte/tranchee/gelee/reportee)
  • Pin - extrait epingle reutilisable (Decisions / criteres / regles)

Strategie & rituels

  • Compass - pilotage strategique (drawer section + outil mental)
  • North Star - intention trimestre/semaine
  • Big Rock - priorite hebdomadaire (max 3 par semaine)
  • Sync - umbrella : Triage matin · Quick Sync hebdo · Evening Sync
  • Triage - rituel matin tri emails + Decisions
  • Pulse - signal faible detecte automatiquement

Voir aussi : VOICE-AICEO.md pour la charte de voix complete (champ navigation maritime + tons par contexte + anti-patterns).
