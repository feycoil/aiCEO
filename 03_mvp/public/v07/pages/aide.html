01Vue d ensemble

aiCEO est votre copilote IA executif local. Il ne remplace pas vos decisions, il les structure et les protege : un cockpit le matin pour arbitrer ce qui compte, des automatismes pour digerer la masse d emails, un rituel le soir pour clore proprement, une revue le vendredi pour ajuster.

Philosophie : 100 % local, SQLite mono-instance, zero cloud applicatif. Vos donnees restent sur votre poste. Anthropic Claude sert uniquement au raisonnement (chat assistant + extraction signaux) - aucune donnee de votre cockpit n est conservee chez Anthropic.

Les 4 cycles

  • Matin (Arbitrage) - 5 a 10 min : trier la file emails, accepter / reporter / ignorer, tirer les decisions du jour.
  • Journee (Cockpit + Actions) : KPIs, projets sous tension, top 3 d aujourd hui, prises de decisions au fil de l eau.
  • Soir (Bilan) - 3 min : humeur, energie, top 3 fait, streak.
  • Semaine (Revue) - 30 min : retrospective, big rocks (max 3) pour la semaine suivante, auto-draft Claude.

05Pieges connus & operations CEO

Cette section est volontairement courte. Pour la liste complete + commandes copiables, ouvrir la page Pilotage (04_docs/00-pilotage-projet.html) onglet "Actions CEO".

Top 3 reflexes

  1. Serveur kazi (port 4747 occupe) : Get-NetTCPConnection -LocalPort 4747 | Stop-Process -Force avant npm start
  2. Reset DB : tuer le serveur d abord, puis cd 03_mvp ; npm run db:reset
  3. Sync Outlook manuelle : pwsh -File scripts/sync-outlook.ps1 (J-15 emails + J-15 a J+30 events)

Si quelque chose casse

  • Logs serveur : dossier logs/ a la racine du repo (rotation 2 Mo / 3 archives)
  • Push Git d urgence : git push origin main --force-with-lease apres backup local
  • Snapshot DB : cp data/aiceo.db data/aiceo.db.bak-$(date +%Y%m%d)

06Glossaire essentiel

  • Big Rocks - les 3 priorites maximum d une semaine. Si tout est prioritaire, rien ne l est.
  • Cap strategique - le "pourquoi" du trimestre. Si la decision du jour ne sert pas le cap, la reporter.
  • Arbitrage matin - rituel de tri 5-10 min : la file d emails est traitee en accepter / reporter / ignorer.
  • Streak - serie de jours consecutifs ou le bilan du soir a ete fait. Effet d ancrage.
  • Atomic Templates - framework v07 : squelette HTML vierge + store unique + ComponentLoader. Garantit la coherence visuelle.
  • Editorial Executive - direction artistique v07 : Reflect + Granola + Linear + Apple HIG. Warm neutrals + accent violet + Aubrielle script (titres).
  • ADR - Architecture Decision Record. Format leger : Statut + Contexte + Decision + Consequences. 30+ ADR dans 00_BOUSSOLE/DECISIONS.md.
  • Lean ADD-AI - methode binome CEO + agent. Sprints livres en 3 a 9 h chrono (vs 13 semaines BASELINE).
