Aide

Commandes & raccourcis

Démarrage

Lance le serveur aiCEO en local sur le port 4747.

cd C:\_workarea_local\aiCEO\03_mvp
npm start

Ou hot-reload : npm run dev

Synchronisation Outlook

Importe vos événements de calendrier et emails depuis Outlook. Lancée auto via schtasks toutes les 2h.

cd C:\_workarea_local\aiCEO\03_mvp
pwsh -File scripts\sync-outlook.ps1

Maintenance base de données

Reset complet de la DB SQLite (efface tout) :

cd C:\_workarea_local\aiCEO\03_mvp
node scripts\init-db.js --reset
npm run seed   # optionnel : peuple avec données démo

Restaurer un backup ou importer des vraies données :

cd C:\_workarea_local\aiCEO
pwsh -File import-real-data.ps1                # auto-détection (backup ou app v0.5)
pwsh -File import-real-data.ps1 -From backup
pwsh -File import-real-data.ps1 -From outlook
pwsh -File import-real-data.ps1 -From status   # juste afficher l'état actuel

Préprod & Release

Lancer un serveur préprod isolé sur port 4748 (base séparée) :

cd C:\_workarea_local\aiCEO
pwsh -File start-preprod.ps1               # lance préprod sur :4748
pwsh -File start-preprod.ps1 -Reset        # reset base préprod avant lancement
pwsh -File start-preprod.ps1 -Public       # + tunnel cloudflare public
pwsh -File stop-preprod.ps1                # arrête préprod (dev :4747 reste)

Créer un ZIP portable pour déploiement externe :

cd C:\_workarea_local\aiCEO
pwsh -File package-release.ps1   # produit release\aiCEO-v0.6-preprod-YYYYMMDD-HHMM.zip

Dépannage

Port 4747 bloqué (EADDRINUSE) :

Get-NetTCPConnection -LocalPort 4747 | ForEach-Object { Stop-Process -Id $_.OwningProcess -Force }

Vérifier la santé du serveur :

Invoke-RestMethod http://localhost:4747/api/system/health

Smoke test complet (12 pages + 4 routes assistant) :

cd C:\_workarea_local\aiCEO\03_mvp
pwsh -File scripts\smoke-all.ps1

Raccourcis clavier dans l'app

  • ⌘K ou Ctrl K · Recherche globale
  • G + H · Aller au cockpit
  • G + A · Aller à l'arbitrage
  • G + P · Aller aux projets
  • G + T · Aller aux tâches
  • Esc · Fermer modale / dropdown
